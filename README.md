# WhereInTheWorld
------------------
This is a game built in javascript
where you get to guess 'Where in the
World' different inventions were 
made!
------------------
# Built using
Node.js
OpenLayers
https://sfbgames.itch.io/chiptone
Natural Earth / Core https://datahub.io/core/geo-countries
